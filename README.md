# 🚀 Talla — طلّة

**"وين الطلّة اليوم؟"**

Talla is a Saudi-first social discovery app that shows you where the vibe is right now.

---

## 📱 What is Talla?

Talla helps users discover the trendiest cafes around them in real time using:

- live check-ins  
- hotspot detection  
- social activity  
- trend scoring (/10)  

---

## 🔥 Key Features

- 📍 Location-based discovery  
- 🔥 Live hotspot ranking  
- ⭐ Trend score system  
- ✅ Real-time check-ins  
- 🏆 Points & streaks  
- 👥 Social profiles & friends  
- 📰 Feed (authentic posts)  
- 💬 Chat  

---

## 🧠 How It Works

Talla transforms real-world locations into **live digital hotspots**.

The more people check in → the hotter the place becomes.
